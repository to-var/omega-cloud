# Translation memory basics

Translation memory stores source and target segment pairs. When you translate a new segment, the tool searches the memory for exact or fuzzy matches.

An exact match means the new source segment is identical to one already in the memory. A fuzzy match means it is similar but not identical; the tool shows a match score (e.g. 85%) and suggests the stored translation so you can adapt it.

Using translation memory reduces repetitive work and keeps terminology consistent across large projects. Most CAT tools support importing and exporting TM data in standard formats like TMX.

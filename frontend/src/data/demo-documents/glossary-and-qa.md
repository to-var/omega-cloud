# Glossary and quality assurance

A project glossary (or termbase) defines how specific terms should be translated. When a term appears in the source, the CAT tool highlights it and suggests the approved translation. This is essential for brand names, technical terms, and legal or medical terminology.

Quality assurance (QA) checks run automatically or on demand. They can flag empty translations, number mismatches, inconsistent terminology, and formatting errors. Fixing issues before delivery saves time and improves client satisfaction.
